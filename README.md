# Crop Recommendation System

This project predicts the most suitable crop to grow based on soil nutrients and environmental conditions using Machine Learning and Flask.

 #Demo [Render Website](https://crop-recommendation-kd43.onrender.com)


# Project Structure
- `app.py` → Flask web app
- `templates/` → HTML templates
- `crop_model.pkl` → Trained ML model
- `label_encoder.pkl` → Label encoder
- `requirements.txt` → Dependencies list

#Tech Stack
- Python
- Flask
- Scikit-learn
- Render (Deployment)
